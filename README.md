# Agenda JS
Uma agenda com login de usuario, adicionar, excluir, editar.

## Package dependencies  

* Dependencies
    * core-js
    * css-loader
    * dotenv
    * ejs
    * express
    * mongoose
    * regenerator-runtime
    * style-loader

* Dev-dependencies
    * webpack
    * nodemon
    * babel

## Getting Started

```sh
# Clone this repository
$ https://github.com/Codfect/Agenda-JS.git

# install dependencies
$ npm i

# Run the project
$ node server.js
$ npm run starwp

```
